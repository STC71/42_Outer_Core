# 🐧 Unix Kernel Projects

Esta carpeta contiene proyectos del currículo de Unix/Kernel de 42 School, enfocados en programación de bajo nivel, arquitectura de sistemas y optimización.

## Proyectos

### [⚙️ libasm](./libasm/) - Assembly Library
Implementación de funciones estándar de C en Assembly x86-64, proporcionando conocimientos profundos sobre arquitectura de sistemas.

**Características destacadas:**
- Funciones de manipulación de strings (`strlen`, `strcpy`, `strcmp`)
- Operaciones de I/O (`read`, `write`)
- Gestión de memoria (`strdup`)
- Bonus: Operaciones de listas enlazadas y conversión de bases
- Rutinas optimizadas en assembly
- Framework completo de testing

**Stack:** Assembly x86-64 (sintaxis Intel), C
