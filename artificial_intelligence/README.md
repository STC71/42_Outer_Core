# 🤖 Artificial Intelligence Projects

Esta carpeta contiene proyectos del currículo de Inteligencia Artificial de 42 School, enfocados en fundamentos de Machine Learning, Data Science y Deep Learning.

## Proyectos

### [📈 ft_linear_regression](./ft_linear_regression/) - Fundamentos de ML
Implementación de regresión lineal desde cero, demostrando los principios básicos del aprendizaje automático.

**Características destacadas:**
- Entrenamiento con descenso de gradiente
- Sistema de predicción
- Visualización de datos y curvas de aprendizaje
- Análisis de precisión (R², MSE, RMSE, MAE, MAPE)

**Stack:** Python, NumPy, Matplotlib

---

### [📊 DSLR](./dslr/) - Data Science × Logistic Regression
Clasificador multiclase usando Regresión Logística con estrategia One-vs-All para clasificar estudiantes de Hogwarts en sus casas.

**Características destacadas:**
- Análisis estadístico completo sin pandas
- Regresión Logística implementada desde cero
- Tres algoritmos de Gradient Descent (Batch, Stochastic, Mini-Batch)
- 99.0% de precisión alcanzada
- Documentación exhaustiva con MATHS.md (2700+ líneas) y PYTHON.md (1500+ líneas)

**Stack:** Python, Matplotlib

---

### [🧠 multilayer-perceptron](./multilayer-perceptron/) - Deep Learning
Red neuronal artificial (Multilayer Perceptron) para clasificación binaria de diagnósticos de cáncer de mama.

**Características destacadas:**
- Red neuronal feedforward desde cero
- Implementación manual de backpropagation
- Mínimo 2 capas ocultas con arquitectura configurable
- Funciones de activación múltiples (Sigmoid, Tanh, ReLU, Softmax)
- Curvas de aprendizaje (Loss y Accuracy)
- Preprocesamiento y normalización de datos
- Evaluación con binary cross-entropy

**Stack:** Python, NumPy, Matplotlib

---

## Progresión del Aprendizaje

Los proyectos están organizados por complejidad creciente:

1. **ft_linear_regression** → Introducción al Machine Learning con regresión simple
2. **DSLR** → Data Science completo con clasificación multiclase
3. **multilayer-perceptron** → Deep Learning con redes neuronales artificiales

Cada proyecto profundiza en conceptos matemáticos, algorítmicos y de implementación, construyendo una base sólida en Inteligencia Artificial.
