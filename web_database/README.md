# 🌐 Web & Database Projects

Esta carpeta contiene proyectos del currículo de Desarrollo Web y Bases de Datos de 42 School, enfocados en aplicaciones full-stack y arquitectura web.

## Proyectos

### [📸 Camagru](./camagru/) - Aplicación Web tipo Instagram
Aplicación web full-stack con captura de fotos desde webcam, edición con stickers y funcionalidades sociales.

**Características destacadas:**
- 📸 Captura desde webcam con getUserMedia API
- 🎨 Superposición de stickers en tiempo real
- 👥 Sistema completo de autenticación (registro, login, verificación email)
- 💬 Interacción social (likes, comentarios con AJAX)
- 📤 Subida de archivos con validación de seguridad
- 📱 Diseño responsive (móvil, tablet, desktop)
- ✨ Bonus: Vista previa en vivo, scroll infinito, GIFs animados

**Stack:** PHP 8.2 (MVC personalizado), MySQL 8.0, JavaScript Vanilla, HTML5/CSS3, Docker

**Documentación:** 19 README detallados + 80+ videos educativos en español
