# 🏠 Vista Home

## 🎯 ¿Qué es y para qué sirve?

La **página de bienvenida** que ves al entrar a Camagru sin estar logueado.

**¿Por qué existe?** Para presentar el proyecto y motivar al usuario a registrarse.

## 📋 Contenido

**`index.php`** - Landing page con presentación del proyecto

## ✨ ¿Qué contiene?

- **Hero section:** Título grande con descripción del proyecto
- **Botones CTA:** "Registrarse" y "Iniciar sesión"
- **Características:** Resumen de qué puede hacer el usuario (webcam, stickers, galería)
- **Diseño responsive:** Se ve bien en cualquier dispositivo
